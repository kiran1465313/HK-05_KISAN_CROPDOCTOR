## 🌍 Problem Statement

Modern agriculture faces a serious challenge:

- 🌱 Nearly **40% of crops are lost every year** due to pests and diseases.
- 💸 Farmers use **30–40% excess pesticides** as a precaution.
- ⚠️ Excess chemicals harm soil fertility, water quality, and human health.
- 🚜 Traditional spraying treats the whole field instead of infected plants.

There is **no affordable system** in the market that:
- Detects infection at individual plant level  
- Determines infection severity  
- Automatically controls pesticide quantity  

---

# 💡 Our Solution – KISAN

**KISAN** is a Smart Intelligent Pesticide Sprinkling System that:

✔ Detects plant infection using sensors and camera  
✔ Analyzes infection level using Machine Learning  
✔ Suggests required pesticide and exact quantity  
✔ Controls an IoT-based smart sprayer  
✔ Provides real-time dashboard monitoring  
✔ Works with edge computing (runs even without internet)  

---

## ⚙️ System Workflow

```text
Plant Monitoring 
      ↓
Infection Detection (Sensors + Camera)
      ↓
AI/ML Analysis
      ↓
Smart Sprayer Activation
      ↓
Real-Time Dashboard Update
```

### 🧠 Edge Computing Advantage
- Faster response  
- No dependency on constant internet  
- Data processed directly on device  
- Immediate prediction output
